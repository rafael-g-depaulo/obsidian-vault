You gotta start up with the Cloud Practitioner Certm and then there are the associate level certs, before the professional ones.

## My personal roadmap
1. Foundational Level
	1. [Cloud Practitioner](https://aws.amazon.com/certification/certified-cloud-practitioner/) (no AI Practitioner because fuck that shit)
2. Associate
	1. [Developer](https://aws.amazon.com/certification/certified-developer-associate/)
	2. [CloudOps Engineer](https://aws.amazon.com/certification/certified-cloudops-engineer-associate/)
	3. (probably) [Solutions Architect](https://aws.amazon.com/certification/certified-solutions-architect-associate/)
	4. (probably not) [Data Engineer](https://aws.amazon.com/certification/certified-data-engineer-associate/)
3. Professional
	1. [DevOps Engineer](https://aws.amazon.com/certification/certified-devops-engineer-professional/)
	2. (probably) [Solutions Architect](https://aws.amazon.com/certification/certified-solutions-architect-professional/)

## Cost
| Certification      | Cost    |
| ------------------ | ------- |
| Foundational Level | 100 USD |
| Associate Level    | 150 USD |
| Professional Level | 300 USD |
